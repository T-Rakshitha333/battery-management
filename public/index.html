Battery Management System API

Use tools like curl, Postman, or your frontend to interact with:

  • POST /api/battery/data – Submit battery data
  • GET /api/battery/:id – Retrieve all data for a battery
  • GET /api/battery/:id/:field – Specific field data
  • GET /api/battery/:id/:field?start=&end= – Field data in time range

🔐 Protected with JWT – use the token in the Authorization header.
